```
UnifiedAIService (Central Hub)
├── AIContentGeneratorService

### 📡 Endpoints da API
```
POST /api/ai/process               # Processamento universal
POST /api/ai/batch                # Processamento em lote

POST /api/ai/generate-content     # Geração de conteúdo
POST /api/ai/analyze-image        # Análise de imagens

POST /api/ai/predict             # Previsões
POST /api/ai/full-analysis       # Análise completa
GET  /api/ai/health              # Health check
