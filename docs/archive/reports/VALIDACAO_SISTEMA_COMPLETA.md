## 💡 INSIGHTS DOS TESTES

### 1. **Correlação Score × Vendas**

Interessante notar que o produto com **mais vendas** (203 unidades) não tem o **maior score** (73/100):

```
Bagageiro Traseiro Moto Universal Suporte Bau
Score: 73/100
Vendas: 203 unidades
```

**Por quê?**
- Produto básico e essencial
- Preço acessível (R$ 89,90)
- Termo "universal" atrai público amplo
- SEO não é o único fator de vendas

**Conclusão:** O sistema de SEO melhora a visibilidade, mas preço, produto e posicionamento também importam.

---

### 2. **Importância dos Contextos**

Produtos com **contextos específicos** têm scores mais altos:

```
Com contextos (delivery, motoboy): 79.2/100
Sem contextos: 68-73/100
```

**Recomendação:** Sempre adicionar contextos de uso nos títulos.

---

### 3. **Tamanho Ideal do Título**

Títulos entre **40-60 caracteres** pontuam melhor:

```
49 caracteres: 84.1/100 (1º lugar)
47 caracteres: 73.0/100 (3º lugar)
38 caracteres: 79.2/100 (2º lugar - salvo pelos contextos)
```

**Recomendação:** Manter entre 40-60 caracteres.
