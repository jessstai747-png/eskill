## 🎓 EXEMPLOS PRÁTICOS

### Use Case 1: E-commerce Product Optimization

```php
use App\Services\SeoService;

$seo = new SeoService();

// Produto original
$product = [
    'title' => 'Tênis Nike',
    'description' => 'Tênis esportivo',
    'price' => 299.90,
    'category' => 'Calçados Esportivos'
];

// Otimizar
$optimized = $seo->optimizeProduct($product);

echo "Título Original: " . $product['title'] . "\n";
echo "Título Otimizado: " . $optimized['optimized_title'] . "\n";
echo "Score SEO: " . $optimized['seo_score'] . "/100\n";

// Resultado:
// Título Original: Tênis Nike
// Título Otimizado: Tênis Nike Air Max Running Masculino Confortável Performance
// Score SEO: 89/100
```

### Use Case 2: Competitor Analysis

```php
use App\Services\CompetitorAnalysisService;

$analyzer = new CompetitorAnalysisService();

// Analisar concorrência
$insights = $analyzer->analyzeCompetitors([
    'product' => 'Notebook Dell Inspiron 15',
    'category' => 'Notebooks',
    'price' => 3500
]);

echo "Concorrentes encontrados: " . count($insights['competitors']) . "\n";
echo "Oportunidades identificadas:\n";
foreach ($insights['opportunities'] as $opp) {
    echo "  • " . $opp . "\n";
}
```

### Use Case 3: Keyword Research

```php
use App\Services\KeywordResearchService;

$research = new KeywordResearchService();

// Pesquisar keywords
$keywords = $research->findKeywords('smartphone', [
    'related' => true,
    'long_tail' => true,
    'limit' => 20
]);

echo "Keywords encontradas:\n";
foreach ($keywords as $kw) {
    $competition = $kw['competition'] > 0.7 ? 'Alta' :
                   ($kw['competition'] > 0.4 ? 'Média' : 'Baixa');

    echo "  • {$kw['keyword']} (Vol: {$kw['volume']}, Comp: {$competition})\n";
}
```
