## 🆕 NOVA FUNCIONALIDADE: Análise Inteligente

### Análise Automática de 391 Itens

```bash
# Analisar todos os itens e gerar plano
php bin/ai-analyze-items.php

# Exportar plano completo em JSON
php bin/ai-analyze-items.php --export

# Exportar CSV para batch processing
php bin/ai-analyze-items.php --csv
```

### O Que o Analyzer Faz

**1. Calcula Score de Prioridade (0-100)**
- Velocidade de vendas (30 pts)
- Qualidade atual do anúncio (25 pts)
- Faixa de preço (20 pts)
- Status do item (15 pts)
- SEO score atual (10 pts)

**2. Estima ROI por Item**
- Views atuais vs projetadas (+145%)
- CTR atual vs projetado (+89%)
- Conversão atual vs projetada (+67%)
- Receita adicional estimada

**3. Identifica Problemas**
- Título muito curto
- Sem descrição
- Ficha técnica incompleta
- Preço não definido
- Item não ativo

**4. Gera Plano Faseado**
- **Fase 1:** Itens Críticos (imediato)
- **Fase 2:** Alta Prioridade (semana 1)
- **Fase 3:** Média Prioridade (semanas 2-3)
- **Fase 4:** Baixa Prioridade (mês 1+)
